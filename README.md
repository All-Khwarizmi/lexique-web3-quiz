# NextJS App Template

Template optimisé pour test technique avec architecture propre et bonnes pratiques.

## 🚀 Quick Start

````bash
# Installation
npm install

# Développement
npm run dev

# Tests
npm run test

# Build
npm run build

## 📁 Structure du Projet

```bash
├── app/          # Routes Next.js App Router
├── core/         # Logique métier
├── infrastructure/   # Implémentation technique
└── ui/          # Composants React
````
